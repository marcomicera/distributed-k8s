# `distributed-k8s`

## `PerfkitBenchmarker`'s Kubernetes example
1. Add the current user to the `docker` group:
    ```bash
    $ newgrp docker
    ```
2. Start the main script:
    ```bash
    $ chmod u+x start.sh
    $ ./start.sh
    ```
