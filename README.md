# `distributed-k8s`

## `PerfkitBenchmarker` + `minikube`
1. Start `minikube` on your local machine:
    ```bash
    $ minikube start
    ```
1. Add the current user to the `docker` group:
    ```bash
    $ newgrp docker
    ```
1. Setup `PerfkitBenchmarker` with Kubernetes:
    ```bash
    $ chmod u+x *.sh
    $ ./setup.sh
    ```
1. Launch `PerfkitBenchmarker`:
    ```bash
    $ ./start.sh
    ```
